# QuickBind вЂ” File Structure

## Technology Stack

| Layer | Technology |
|-------|-----------|
| App framework | Tauri v2 |
| Frontend | React 18 + TypeScript |
| Build tool | Vite 5 |
| Styling | Tailwind CSS v3 + CSS variables |
| Components | Radix UI primitives |
| Icons | Lucide React |
| Fonts | Geist + Geist Mono (@fontsource) |
| State | Zustand |
| Backend | Rust (Tauri core) |
| Shortcuts | tauri-plugin-global-shortcut |
| Tray | Tauri built-in (tray-icon feature) |
| Autostart | tauri-plugin-autostart |
| Storage | tauri-plugin-store (JSON file, local) |
| Animations | tailwindcss-animate |

## Key File Details

### `src/types/index.ts`
```typescript
export type ActionType = 'launch_app' | 'open_url' | 'run_script' | 'type_text';

export interface Shortcut {
  id: string;           // UUID
  name: string;         // Display name
  keys: string[];       // e.g. ["Ctrl", "Shift", "K"]
  action_type: ActionType;
  action_value: string; // path, url, script, or text
  description?: string;
  enabled: boolean;
  created_at: number;   // unix timestamp
}

export interface AppSettings {
  autostart: boolean;
  minimize_to_tray: boolean;
  show_notifications: boolean;
}
```

### `src-tauri/src/models.rs`
```rust
use serde::{Deserialize, Serialize};

#[derive(Debug, Serialize, Deserialize, Clone)]
pub struct Shortcut {
    pub id: String,
    pub name: String,
    pub keys: Vec<String>,
    pub action_type: String,
    pub action_value: String,
    pub description: Option<String>,
    pub enabled: bool,
    pub created_at: u64,
}

#[derive(Debug, Serialize, Deserialize, Clone)]
pub struct AppSettings {
    pub autostart: bool,
    pub minimize_to_tray: bool,
    pub show_notifications: bool,
}
```

## Data Storage

Shortcuts and settings are stored via `tauri-plugin-store` in a local JSON file:

```
%APPDATA%\com.quickbind.app\quickbind.json
```

Structure:
```json
{
  "shortcuts": [ ...Shortcut[] ],
  "settings": { ...AppSettings }
}
```

No cloud. No accounts. Everything local.
